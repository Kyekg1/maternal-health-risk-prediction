# maternal-health-risk-prediction
This repository contains the formal research proposal, methodology design and preliminary architectures for developing predictive models to identify ,maternal risks. The goal of this project is to leverage machine learning algorithms to detect early warning signs of complications during pregnancy and postpartum phases
